### 選單介紹
* 學校資訊：跳轉學校網站。
* 臺東天氣：串接API，可抓取其他縣市天氣。

<div align=center>
    <img width="600" height="200" src="https://i.imgur.com/ln8yXhq.png"/>
</div>

* 圖資館：臺東大學圖書館常用服務。
* 學校公告：爬取學校前五項最新公告及提供公告網址。
 * 抽學餐：可再抽一次並提供菜單連結。
 * 快速選單：
    * 8129公車
    * 學生會官網
    * 學生會IG
    * 學校電話
    * 宿舍
    * 校園地圖
    * 課程資訊
    * 意見回饋
    * 尬聊
    * 操作說明
